Summary
===
This project contains the result of the course project 1 for the EDA class.

It contains 8 files:

- plot1.R produces plot1.png  
- plot2.R produces plot2.png  
- plot3.R produces plot3.png  
- plot4.R produces plot4.png  


Instructions
===

Download the file from the following link into your working directory: [link]"https://d396qusza40orc.cloudfront.net/exdata%2Fdata%2Fhousehold_power_consumption.zip"

Download the 4 plot*.R files from this github repository to your working directory.

Run each of the four plot scripts to produce the corresponding png graph.
